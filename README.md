##  YouTube Clone ‚Äì MERN Stack Project

This project is a **YouTube Clone** built using the **MERN Stack (MongoDB, Express, React, Node.js)** as part of a Full Stack Development Capstone Project.

It supports **user authentication**, **channel creation**, **video upload & playback**, **likes/dislikes**, **search**, and **responsive UI** similar to YouTube.

---

##  Tech Stack

### Frontend
- React (Vite)
- React Router DOM
- Axios
- CSS (Responsive Grid Layout)

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt.js
- dotenv

---

##  Features

-  User Registration & Login (JWT)
-  Channel creation and viewing
-  Video upload & playback
-  Like & Dislike videos
-  Search videos by title
-  MVC-based backend architecture
-  Fully responsive UI with grid layout
