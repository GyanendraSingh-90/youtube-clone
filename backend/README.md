This is the **backend** of the YouTube Clone project built using **Node.js, Express, and MongoDB** as part of a Full Stack MERN Capstone Project.  
It provides RESTful APIs for authentication, channels, videos, likes/dislikes, comments, and search functionality.

---

##  Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB + Mongoose**
- **JWT (Authentication)**
- **bcrypt.js**
- **dotenv**

---

##  Features

-  User Registration & Login with JWT
-  Channel creation and retrieval
-  Video upload and fetch APIs
-  Like & Dislike functionality
-  Search videos by title and category
-  Comment system
-  Modular MVC architecture

---

## Installation & Setup
1. Clone the repository

2. Install dependencies
npm install

3. Start MongoDB
Make sure MongoDB is running locally or use MongoDB Atlas.

4. Start the server
npm start
